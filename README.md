## About this repo

This repo holds the LaTeX sources of my CV plus the latest compiled PDF. The compiled PDF is what you want if you only need to read or download the CV.

## Versions

| File | Style | Status |
|---|---|---|
| `cv.pdf` | Harvard-style template — clean, well-spaced, single column | ✅ Compiled, **current version** |
| `latex/cv_harvard.tex` | Source for the PDF above | ✅ Ready to compile |
| `latex/cv_modern.tex` | Variant with bold-prefixed bullets and 2-column skills | 📝 Alternate source — kept for reference |

The Harvard template is the canonical version: it leaves clear separation between sections, gives every entry room to breathe, and reads cleanly as a printed document. The modern variant (`cv_modern.tex`) was an experiment in bold-prefixed bullets — it is kept in the repo as an alternate source but is not the recommended layout.

## How to compile

The Harvard CV compiles cleanly in **Overleaf** (recommended) or with a local TeX Live install:

```bash
pdflatex latex/cv_harvard.tex
```
